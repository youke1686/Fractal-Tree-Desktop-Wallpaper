# 프랙탈 트리 데스크톱 배경화면

[中文](README.md) | [English](README_en.md) | [日本語](README_ja.md) | [한국어](README_ko.md) | [Русский](README_ru.md) | [Français](README_fr.md) | [Deutsch](README_de.md) | [Español](README_es.md)

C# Windows Forms로 개발된 프랙탈 트리 생성기로, 다양한 형태의 프랙탈 트리를 생성하여 Windows 데스크톱 배경화면으로 설정할 수 있습니다.
컴퓨터 데스크톱에서 나무를 심어보세요!

## 소개

`프랙탈 트리 데스크톱 배경화면`은 재귀 알고리즘과 매개변수 설정을 통해 세상의 거의 모든 종류의 나무 형태를 시뮬레이션할 수 있는 강력한 프랙탈 트리 생성 도구입니다.

## 주요 기능

- **강력한 매개변수 시스템**: 정말 강력합니다
- **프리셋 트리 라이브러리**: 매개변수 조정을 못해도 문제없습니다
- **사용자 정의 편집**: 사용자 정의 트리 종의 생성, 편집, 저장, 가져오기, 내보내기 지원
- **다국어 지원**: 8개 언어 지원 (단, 중국어를 제외한 모든 언어는 AI 번역입니다)
- **원클릭 내보내기/배경화면 설정**: 식수 효율 향상
- **낙엽 시스템**: 분위기 만점

## 설치

1. [프로젝트 릴리스 페이지](https://github.com/youke1686/Fractal-Tree-Desktop-Wallpaper/releases) 방문
2. 장치에 맞는 최신 버전의 `exe` 파일 다운로드
3. 설치 불필요, 더블 클릭으로 시작

## 빠른 시작

1. 프로그램 실행
2. 메인 화면에서 관심 있는 나무를 선택하고 클릭하여 효과 확인
3. "새로고침"을 클릭하여 다른 나무 표시
4. 이미지로 내보내기 또는 배경화면으로 설정
