# Fraktaler Baum - Desktop-Hintergrundbild

[中文](README.md) | [English](README_en.md) | [日本語](README_ja.md) | [한국어](README_ko.md) | [Русский](README_ru.md) | [Français](README_fr.md) | [Deutsch](README_de.md) | [Español](README_es.md)

Ein mit C# Windows Forms entwickelter Generator für fraktale Bäume, der verschiedene Formen von fraktalen Bäumen generieren und als Windows-Desktop-Hintergrundbilder festlegen kann.
Lassen Sie uns Bäume auf Ihrem Computer-Desktop pflanzen!

## Einführung

`Fraktaler Baum - Desktop-Hintergrundbild` ist ein leistungsstarkes Werkzeug zur Generierung fraktaler Bäume, das durch rekursive Algorithmen und parametrische Konfiguration fast alle Baumformen der Welt simulieren kann.

## Hauptfunktionen

- **Leistungsstarkes Parametersystem**: wirklich leistungsstark
- **Voreingestellte Baum-Bibliothek**: kein Problem, wenn Sie die Parameter nicht anpassen können
- **Benutzerdefinierte Bearbeitung**: Unterstützung beim Erstellen, Bearbeiten, Speichern, Importieren und Exportieren benutzerdefinierter Baumarten
- **Mehrsprachige Unterstützung**: unterstützt 8 Sprachen (obwohl alle außer Chinesisch KI-übersetzt sind)
- **Ein-Klick-Export/Hintergrundbild-Festlegung**: Effizienz beim Baumpflanzen steigern
- **Laub-System**: voller Atmosphäre

## Installation

1. Besuchen Sie die [Projekt-Veröffentlichungsseite](https://github.com/youke1686/Fractal-Tree-Desktop-Wallpaper/releases)
2. Laden Sie die neueste Version der `exe`-Datei herunter, die zu Ihrem Gerät passt
3. Keine Installation erforderlich, einfach doppelklicken zum Starten

## Schnellstart

1. Führen Sie das Programm aus
2. Wählen Sie einen interessanten Baum auf der Hauptoberfläche und klicken Sie, um den Effekt zu sehen
3. Klicken Sie auf "Aktualisieren", um einen anderen Baum zu erhalten
4. Als Bild exportieren oder als Hintergrundbild festlegen
