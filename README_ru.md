# Фрактальное дерево - Обои для рабочего стола

[中文](README.md) | [English](README_en.md) | [日本語](README_ja.md) | [한국어](README_ko.md) | [Русский](README_ru.md) | [Français](README_fr.md) | [Deutsch](README_de.md) | [Español](README_es.md)

Генератор фрактальных деревьев, разработанный на C# Windows Forms, который может генерировать различные формы фрактальных деревьев и устанавливать их в качестве обоев рабочего стола Windows.
Давайте сажать деревья на рабочем столе компьютера!

## Введение

`Фрактальное дерево - Обои для рабочего стола` - это мощный инструмент генерации фрактальных деревьев, который с помощью рекурсивных алгоритмов и параметрической конфигурации может моделировать почти все типы деревьев в мире.

## Основные функции

- **Мощная система параметров**: действительно мощная
- **Библиотека предустановленных деревьев**: не проблема, если не умеете настраивать параметры
- **Пользовательское редактирование**: поддержка создания, редактирования, сохранения, импорта и экспорта пользовательских видов деревьев
- **Многоязычная поддержка**: поддержка 8 языков (хотя, кроме китайского, все переведены ИИ)
- **Экспорт/установка обоев в один клик**: повышение эффективности посадки деревьев
- **Система опавших листьев**: полная атмосфера

## Установка

1. Посетите [страницу релизов проекта](https://github.com/youke1686/Fractal-Tree-Desktop-Wallpaper/releases)
2. Скачайте последнюю версию файла `exe`, соответствующего вашему устройству
3. Установка не требуется, просто дважды щелкните для запуска

## Быстрый старт

1. Запустите программу
2. Выберите интересующее дерево в главном окне и нажмите для просмотра эффекта
3. Нажмите "Обновить" для получения другого дерева
4. Экспортируйте как изображение или установите в качестве обоев
