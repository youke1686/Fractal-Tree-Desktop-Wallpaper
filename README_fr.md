# Arbre Fractal - Fond d'Écran de Bureau

[中文](README.md) | [English](README_en.md) | [日本語](README_ja.md) | [한국어](README_ko.md) | [Русский](README_ru.md) | [Français](README_fr.md) | [Deutsch](README_de.md) | [Español](README_es.md)

Un générateur d'arbres fractals développé avec C# Windows Forms qui peut générer diverses formes d'arbres fractals et les définir comme fonds d'écran de bureau Windows.
Plantons des arbres sur le bureau de votre ordinateur !

## Introduction

`Arbre Fractal - Fond d'Écran de Bureau` est un puissant outil de génération d'arbres fractals qui, grâce à des algorithmes récursifs et une configuration paramétrique, peut simuler presque tous les types de formes d'arbres dans le monde.

## Fonctionnalités Principales

- **Système de Paramètres Puissant**: vraiment puissant
- **Bibliothèque d'Arbres Prédéfinis**: pas de problème si vous ne savez pas ajuster les paramètres
- **Édition Personnalisée**: support pour créer, éditer, sauvegarder, importer et exporter des espèces d'arbres personnalisées
- **Support Multilingue**: supporte 8 langues (bien que, sauf le chinois, toutes soient traduites par IA)
- **Export/Définition de Fond d'Écran en Un Clic**: améliorer l'efficacité de plantation d'arbres
- **Système de Feuilles Mortes**: pleine d'atmosphère

## Installation

1. Visitez la [page de publication du projet](https://github.com/youke1686/Fractal-Tree-Desktop-Wallpaper/releases)
2. Téléchargez la dernière version du fichier `exe` correspondant à votre appareil
3. Pas d'installation nécessaire, double-cliquez simplement pour démarrer

## Démarrage Rapide

1. Exécutez le programme
2. Sélectionnez un arbre intéressant dans l'interface principale et cliquez pour voir l'effet
3. Cliquez sur "Rafraîchir" pour obtenir un arbre différent
4. Exportez en image ou définissez comme fond d'écran
