# フラクタルツリー・デスクトップ壁紙

[中文](README.md) | [English](README_en.md) | [日本語](README_ja.md) | [한국어](README_ko.md) | [Русский](README_ru.md) | [Français](README_fr.md) | [Deutsch](README_de.md) | [Español](README_es.md)

C# Windows Formsで開発されたフラクタルツリー生成ツールで、様々な形のフラクタルツリーを生成し、Windowsデスクトップの壁紙として設定できます。
パソコンのデスクトップで植樹しましょう！

## 紹介

`フラクタルツリー・デスクトップ壁紙`は、再帰アルゴリズムとパラメータ設定により、世界中のほぼすべての種類の木の形をシミュレートできる強力なフラクタルツリー生成ツールです。

## 主な機能

- **強力なパラメータシステム**：本当に強力です
- **プリセットツリー庫**：パラメータの調整ができなくても問題ありません
- **カスタム編集**：カスタムツリー種の作成、編集、保存、インポート、エクスポートをサポート
- **多言語サポート**：8言語をサポート（ただし、中国語以外はすべてAI翻訳です）
- **ワンクリックエクスポート/壁紙設定**：植樹効率を向上
- **落葉システム**：雰囲気満点

## インストール

1. [プロジェクトリリースページ](https://github.com/youke1686/Fractal-Tree-Desktop-Wallpaper/releases)にアクセス
2. お使いのデバイスに対応する最新版の`exe`ファイルをダウンロード
3. インストール不要、ダブルクリックで起動

## クイックスタート

1. プログラムを実行
2. メイン画面で興味のある木を選択し、クリックして効果を確認
3. 「更新」をクリックして別の木を表示
4. 画像としてエクスポートまたは壁紙として設定
