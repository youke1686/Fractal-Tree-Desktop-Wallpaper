# 分形树桌面壁纸

一款基于 C# Windows Forms 开发的分形树生成器，可以生成各种形态的分形树并设置为 Windows 桌面壁纸。
一起在电脑桌面植树造林吧！

## 项目简介

`分形树桌面壁纸` 是一个功能强大的分形树生成工具，通过递归算法和参数化配置，能够模拟世界上几乎所有类型的树木形态。

## 核心特性

- **强大的参数系统**：反正就是很强大
- **预设树种库**：不会调参数也没关系
- **自定义编辑**：支持创建、编辑、保存、导入、导出自定义树种
- **多语言支持**：支持 8 种语言（虽然除了中文都是AI翻译的）
- **一键导出/设置壁纸**：提高种树效率
- **落叶系统**：氛围感拉满

## 安装

1. 访问[项目发布页](https://github.com/youke1686/Fractal-Tree-Desktop-Wallpaper/releases)
2. 下载最新版本与你设备匹配的 `exe` 文件
3. 无需安装，双击即可启动

## 快速开始

1. 运行程序
2. 在主界面选择一个感兴趣的树，点击查看效果
3. 点击"刷新"换一棵树
4. 导出为图片或设为壁纸
