# 프랙탈 트리 데스크톱 월페이퍼 - 사용 가이드

> **참고**: 이 문서는 AI가 원본 중국어 버전에서 번역했습니다. 오류나 불명확한 표현이 있으면 원본 `help.md` 파일을 참조해 주세요.

## 소개

`프랙탈 트리 데스크톱 월페이퍼`는 아름다운 나무를 무작위로 생성할 수 있습니다. 강력한 파라미터 시스템 덕분에 **자연에 존재하는 나무**부터 **상상 속의 나무**까지 전 세계의 **거의 모든 종류**의 나무를 시뮬레이션할 수 있습니다.

### `프랙탈 트리`란 무엇인가?

`프랙탈 트리`는 **재귀적으로 생성**되는 나무 구조입니다. 각 가지는 여러 하위 가지를 가지고, 각 하위 가지는 또 다시 여러 하위 가지를 포함하며, 이를 반복하여 나무와 같은 구조를 형성합니다.  
`프랙탈 트리 데스크톱 월페이퍼`에서는 렌더링 시 계산량을 줄이고 제어성을 높이기 위해 각 가지가 최대 2개의 하위 가지만 가지는 이진 트리만을 사용합니다.

가능한 한 현실적인 나무 구조를 구현하기 위해 `프랙탈 트리 데스크톱 월페이퍼`는 노드 간의 거리, 각도, 선 색상 등을 제어하는 다양한 파라미터를 제공합니다.

## 파라미터 입문

`프랙탈 트리 데스크톱 월페이퍼`를 처음 사용하시는 분들은 많은 파라미터에 겁먹지 마세요. 몇 가지 핵심 파라미터만 조정하면 독특한 나무를 만들 수 있습니다. `분기 각도 범위`와 `길이 감쇠 범위`를 수정하여 나무의 형태가 어떻게 변하는지 확인해 보세요. 그런 다음 `잎 설정`의 **색상**을 조정하여 나무에 색을 더해 보세요. 기본 작업에 익숙해지면 **파라미터 그룹**과 **분포 곡선**을 더 탐색하여 각 나무를 진정한 예술 작품으로 만들 수 있습니다.

다음은 기본 파라미터 튜토리얼입니다 ↓

### 1. 분기 각도 범위

`분기 각도 범위`는 부모 가지에 대한 자식 가지의 편향 각도 범위(도)를 제어합니다. 분포 곡선을 통해 범위 내에서 각도 값의 분포 확률을 제어합니다(분포 곡선은 기본 튜토리얼 범위를 벗어나므로 기본 균등 분포를 유지하면 됩니다).

- `분기 각도 범위`가 **전체적으로 크면** 수관은 **드문드문하고 볼륨감 있게** 보입니다.
- `분기 각도 범위`가 **전체적으로 작으면** 수관은 **빽빽하게** 보입니다.
- `분기 각도 범위`의 **구간이 넓으면** 수관은 **무질서하게** 보입니다.
- `분기 각도 범위`의 **구간이 좁으면** 수관은 **정돈되게** 보입니다.

### 2. 길이 감쇠 범위

`길이 감쇠 범위`는 부모 가지 길이에 대한 자식 가지 길이의 비율 범위를 제어합니다. 분포 곡선을 통해 범위 내에서 비율 값의 분포 확률을 제어합니다(이것도 기본 튜토리얼 범위를 벗어나므로 기본 균등 분포를 유지하면 됩니다).

예를 들어, 한 번의 분기에서 길이 감쇠가 사전 설정 범위에서 **0.7**이 무작위로 선택되면, 자식 가지 길이는 부모 가지 길이의 **70%**가 됩니다.

- `길이 감쇠 범위`가 **전체적으로 크면** 수관은 **거대하게** 보입니다.
- `길이 감쇠 범위`가 **전체적으로 작으면** 수관은 **작고 드문드문하게** 보입니다.
- `길이 감쇠 범위`의 **구간이 넓으면** 수관은 **층층이 다양하게** 보입니다.
- `길이 감쇠 범위`의 **구간이 좁으면** 수관은 **균일하게 빽빽하게** 보입니다.

> 주의: 특별한 요구 사항이 없다면 `길이 감쇠 범위`의 최대값은 **0.95**를 초과하지 않는 것이 좋습니다. 그렇지 않으면 수관이 너무 빽빽해지거나 무한히 성장하여(`최대 깊이`로만 제한 가능) 렌더링 성능에 영향을 줄 뿐만 아니라 수관이 극히 부자연스럽게 보입니다.

### 3. 잎 설정

`잎 설정`은 잎의 형태를 제어합니다. 잎 색상, 가중치, 밀도, 선 너비를 제어하여 다양한 잎 효과를 구현합니다.(기본 튜토리얼에서는 색상만 수정하면 됩니다.)

먼저 약간의 차이가 있는 여러 녹색을 동시에 추가하여 효과를 확인해 보세요.

### 4. 배경색

이름 그대로 `배경색`은 캔버스의 배경색을 제어합니다.  
연한 잎 색조를 사용하면 더 자연스럽게 보입니다.

더 전문적인 색상 제안을 원하시면 [컬러 휠 도구 | Adobe Express](https://color.adobe.com/create/color-wheel)를 참조하세요.

## 파라미터 심화

이 섹션에서는 각 파라미터의 상세한 정의와 사용법을 설명합니다.

픽셀 단위의 모든 파라미터는 창 높이가 표준 1080픽셀일 때의 길이를 나타냅니다. 실제 렌더링 시 창 높이에 따라 비례하여 스케일링됩니다.

---

### 1. 글로벌 설정

글로벌 설정은 파라미터 그룹에 포함되지 않고 효과에 직접 영향을 줍니다.

#### 1.1 나무 이름

`나무 이름`은 나무에 이름을 붙이는 데 사용합니다.  
메인 화면 목록에 표시되며, 내보내기 이미지의 기본 파일명 접두사로 사용됩니다. 빈 문자열은 자동으로 **"무제"**로 수정됩니다.

#### 1.2 배경색

`배경색`은 캔버스의 배경색을 제어하는 데 사용합니다.  
창 배경색과 월페이퍼 이미지 배경색 채우기에 사용됩니다.

---

### 2. 기본 형태 파라미터

다음 파라미터는 나무의 기본 형태와 종료 조건을 직접 제어합니다.

#### 2.1 줄기 길이

`줄기 길이`는 뿌리에서 발하는 첫 번째 레벨 가지의 초기 길이(픽셀)입니다. 뿌리는 캔버스 하단에서 10% 위쪽에 위치합니다.  
나무 전체의 기준 스케일을 결정하며, 나무의 최종 크기에 직접 영향을 줍니다.

#### 2.2 최소 가지 길이

`최소 가지 길이`는 나무의 종료 조건 중 하나입니다. 가지 길이가 이 임계값보다 낮으면 하위 가지가 생성되지 않습니다.  
수관 끝의 세밀함과 수관의 크기를 제어합니다.

#### 2.3 최대 깊이

`최대 깊이`는 나무의 종료 조건 중 하나입니다. 가지의 재귀 레벨(뿌리는 레벨 0, 뿌리에서 분기한 자식 가지는 레벨 1, 이하 동일)이 이 값에 도달하면 분기가 중지됩니다.  
수관이 너무 빽빽해지거나 무한히 성장하여 성능에 영향을 주거나 애플리케이션이 멈추는 것을 방지합니다.

> 주의: 창작 초기에는 `최대 깊이`를 **10~30**으로 설정하고, 파라미터가 적절하지 않아 테스트 중 멈추는 것을 방지하기 위해 나중에 필요에 따라 조정하세요.

#### 2.4 잎 임계값

`잎 임계값`은 현재 가지를 가지로 렌더링할지 잎으로 렌더링할지 결정하는 임계값입니다. 가지 길이에 따라 시각적 역할을 구분합니다. 현재 가지 길이가 이 임계값보다 낮으면 해당 가지는 잎으로 렌더링되고, 임계값보다 높으면 줄기로 렌더링됩니다.  
잎의 시각적 효과를 제어하며, 수관의 형태에 영향을 줍니다.

> 일반적으로 잎 밀도를 직접 조정하는 데 사용하는 것은 권장하지 않습니다. 이는 잎의 형태와 관련이 있으므로 다른 파라미터를 조정하여 구현해야 합니다.

#### 2.5 최대 가지 기울기

`최대 가지 기울기`는 나무의 종료 조건 중 하나입니다. 자식 가지의 절대 각도(수직 위쪽 방향에 대한 각도)의 상한(도)을 제어합니다. 이 값을 초과하는 자식 가지는 생성되지 않고 직접 종료됩니다.  
가지가 아래로 처지는 것을 방지하고 나무의 형태를 유지합니다.

---

### 3. 분기 각도 범위

`분기 각도 범위`는 각 자식 가지가 독립적으로 샘플링하는 편향 각도 구간 범위입니다. 두 자식 가지는 각각 부모 가지의 양쪽으로 편향됩니다. 샘플링 범위는 [최소값, 최대값](도)입니다.  
가지의 형태를 제어하며, 가지의 정돈됨과 성장 방식에 영향을 줍니다.

#### 3.1 각도 확률 `분포 곡선`

`분포 곡선` 버튼으로 막대 그래프 편집기를 엽니다: 막대를 클릭하여 막대 높이를 조정합니다. 높이 범위는 `0.02 ~ 1.0`입니다.

**각 자식 가지의 샘플링 방식**: 각도 구간을 8개의 하위 구간으로 균등 분할하고, 먼저 분포 곡선의 막대 그래프 높이를 가중치로 하위 구간을 선택한 후, 해당 하위 구간 내에서 균등하게 무작위 값을 취합니다. 분포 곡선이 지정되지 않으면 전체 구간에서 균등하게 무작위 값을 취합니다.

간단히 말해, 막대가 높을수록 그 부근의 값이 가지 각도로 선택될 확률이 높아집니다.

#### 3.2 참고 예시

기본 나무 종 `소나무`의 분기 각도 범위는 `[0, 90]`이며, 각도 확률 분포 곡선은 대략 다음과 같습니다:

<!-- BAR_CHART: [1.00, 0.02, 0.02, 0.02, 0.02, 0.02, 0.02, 0.20] | [0°, 12.86°, 25.71°, 38.57°, 51.43°, 64.29°, 77.14°, 90°] -->

소나무의 분기 각도가 0도와 90도 부근을 취할 확률이 높은 것을 볼 수 있습니다. 즉, 소나무의 가지는 사선으로 성장하는 것이 아니라 부모 가지에 거의 평행하거나 수직이 될 확률이 높습니다.  
실제 효과는 소나무가 대부분 곧게 성장하지만 때로는 크게 방향을 틀어, 현실의 소나무의 굽고 힘찬 형태와 유사한 것을 제공합니다.  
부모 가지에 평행할 확률이 더 높은 것은 수관의 기본 형태가 너무 혼란스럽지 않도록 유지하기 위함입니다.

### 4. 길이 감쇠 범위

`길이 감쇠 범위`는 각 자식 가지가 독립적으로 샘플링하는 길이 감쇠 비율 구간 범위입니다. 자식 가지의 실제 길이 = 부모 가지 길이 × 샘플링된 감쇠 비율입니다. 샘플링 범위는 [최소값, 최대값](감쇠 비율)입니다.

#### 4.1 감쇠 확률 `분포 곡선`

[3.1 각도 확률 분포 곡선](#31-각도-확률-분포-곡선)과 유사합니다.

`분포 곡선` 버튼으로 막대 그래프 편집기를 엽니다: 막대를 클릭하여 막대 높이를 조정합니다. 높이 범위는 `0.02 ~ 1.0`입니다.

**각 자식 가지의 샘플링 방식**: 각도 구간을 8개의 하위 구간으로 균등 분할하고, 먼저 분포 곡선의 막대 그래프 높이를 가중치로 하위 구간을 선택한 후, 해당 하위 구간 내에서 균등하게 무작위 값을 취합니다. 분포 곡선이 지정되지 않으면 전체 구간에서 균등하게 무작위 값을 취합니다.

간단히 말해, 막대가 높을수록 그 부근의 값이 감쇠 비율로 선택될 확률이 높아집니다.

#### 4.2 참고 예시

기본 나무 종 `녹색 나무`의 길이 감쇠 범위는 `[0.6, 0.94]`이며, 감쇠 확률 분포 곡선은 대략 다음과 같습니다:

<!-- BAR_CHART: [0.84, 0.02, 0.02, 0.02, 0.02, 0.02, 0.02, 0.70] | [0.60, 0.65, 0.70, 0.75, 0.79, 0.84, 0.89, 0.94] -->

녹색 나무의 감쇠 비율이 0.60과 0.94 부근을 취할 확률이 높은 것을 볼 수 있습니다. 즉, 녹색 나무의 가지는 부모 가지와 거의 같은 길이이거나 명확히 짧습니다.  
실제 효과는 녹색 나무가 때로는 가지가 펼쳐지고 때로는 잎을 모아, 층층이 다양한 형태를 제공합니다.  
녹색 나무가 너무 이르게 짧은 가지를 선택하면 나무 전체의 높이가 직접 감소하여 나무 높이의 차이가 커진다는 점에 주의해야 합니다.

### 5. 줄기/잎 설정

`줄기/잎 설정`은 줄기와 잎의 시각적 효과를 결정하는 파라미터 집합으로, 나무의 구조에는 영향을 주지 않습니다.

줄기와 잎은 각각 독립적인 색상 목록을 가집니다. 각 색상 항목은 세 가지 속성을 포함합니다:

#### 5.1 색상

`색상`은 줄기 또는 잎 렌더링에 사용할 색상을 결정하는 파라미터입니다. 줄기/잎에 여러 색상을 설정할 수 있으며 무작위로 선택됩니다.

#### 5.2 가중치

`가중치`는 새 줄기 또는 잎을 그릴 때 이 색상이 선택될 확률을 결정하는 파라미터입니다. 값이 클수록 해당 색상의 줄기/잎에서의 비율이 높아집니다.

#### 5.3 선 너비 비율

`선 너비 비율`은 줄기 또는 잎의 너비 계수를 결정하는 파라미터입니다. 값이 클수록 줄기/잎이 굵어집니다. 실제 굵기는 가지 길이에도 좌우됩니다.

**다중 파라미터 그룹 동작**: 줄기/잎 설정은 다중 파라미터 그룹의 수치 가중 평균에 참여하지 않습니다. 여러 파라미터 그룹이 존재하면 가중치가 가장 큰 그룹의 줄기/잎 설정을 직접 사용합니다.

### 6. 낙엽 시스템

`낙엽 시스템`은 지면에 낙엽을 추가하는 기능입니다. 잎이 그려진 후 지면 영역에 동일한 형태의 낙엽이 추가로 그려질 확률이 있습니다.  
나무의 분위기를 높입니다.

#### 6.1 낙엽 밀도

`낙엽 밀도`는 잎이 지면 낙엽으로 변환될 확률을 결정하는 파라미터입니다. 값 범위는 `0.0 ~ 1.0`입니다. 값이 클수록 낙엽 수가 많아집니다.  
실제 낙엽 수는 수관의 밀도(나무 위의 잎 수)에도 좌우됩니다.

#### 6.2 낙엽 분산도

`낙엽 분산도`는 낙엽의 분포 범위를 결정하는 파라미터입니다. 구체적으로는 원래 잎에서의 수평 오프셋 범위(픽셀)와 지면 위치에서의 수직 오프셋 범위(픽셀)를 가리킵니다. 값이 클수록 낙엽의 분포 범위가 넓어집니다.

지면 위치에서의 수직 오프셋은 삼각 분포(두 균등 무작위 값의 평균)를 사용합니다.  
원래 잎에서의 수평 오프셋은 균등 분포를 사용합니다. 다수의 잎의 무작위성이 이미 정규 분포와 유사한 효과를 달성하기 때문입니다.

### 7. 파라미터 그룹 시스템

`파라미터 그룹`은 `프랙탈 트리 데스크톱 월페이퍼`의 핵심 추상 개념입니다. 단일 나무 종 설정은 하나 이상의 파라미터 그룹을 포함할 수 있으며, 각 그룹은 독립적인 파라미터 세트를 가집니다.

#### 7.0 파라미터 그룹이 필요한 이유

상상해 보세요. 줄기는 곧게 뻗고 수관은 펼쳐진 나무를 디자인하고 싶다고 가정합시다.  
하나의 파라미터 세트만 사용하면 줄기와 수관의 형태가 같을 수밖에 없어 복잡한 효과를 구현할 수 없습니다.

사실 이것은 `단풍나무`를 디자인할 때 제가 실제로 겪었던 문제입니다.  
이를 위해 저는 두 가지 아이디어를 생각했습니다:

첫 번째 아이디어는 가지의 y 좌표를 "파라미터의 파라미터"로 통합하여 높이에 따라 파라미터가 변화하도록 직접 구현하는 것입니다. 즉, 파라미터를 y의 함수로 작성할 수 있습니다.  
이것은 프로그래머의 사고에 부합하며 더 복잡하고 정확한 형태 제어를 구현할 수 있습니다. 하지만 분명히 직관적이지 않습니다. 실제 작업 시 상당히 복잡한 추상적 사고가 필요하며, 계산기나 Desmos 같은 도구를 사용하여 함수 그래프를 연구하고 파라미터를 반복적으로 조정해야 합니다.

그래서 저는 두 번째 아이디어인 파라미터 그룹 시스템을 선택했습니다.  
여러 파라미터 세트를 직접 정의하고 서로 다른 높이에서 서로 다른 파라미터 그룹을 적용하여 줄기와 수관의 형태를 구분합니다.

실제 개발 시 자연스러운 전환을 위해 서로 다른 파라미터 그룹의 파라미터를 역가중 혼합하여 적용하기로 했습니다.

#### 7.1 기본 작업

`다중 파라미터 그룹 제어`에서:

- **드롭다운**: 편집할 파라미터 그룹을 선택합니다. 클릭하여 파라미터 그룹 이름을 수정할 수 있습니다.
- **「+」**: 새 파라미터 그룹을 추가합니다(현재 선택된 파라미터 그룹의 파라미터 복사).
- **「✕」**: 현재 선택된 파라미터 그룹을 삭제합니다.

#### 7.2 대표적 유효 높이

`대표적 유효 높이`는 이 파라미터 그룹의 앵커 높이(픽셀)로, 다중 파라미터 그룹의 가중 혼합 계산에 사용됩니다.  
파라미터 그룹이 효과를 발휘하는 대략적인 위치를 제어합니다. 가지가 지면에서의 높이가 대표적 유효 높이에 가까울수록 파라미터 그룹의 가중치가 높아집니다.

간단히 말해, 파라미터는 `대표적 유효 높이` 부근에서 효과를 발휘합니다.

#### 7.3 가중치 계산

가지가 높이 $h$에 있을 때, 그룹 $i$의 원래 가중치는:

$$w_i = \min\left(\frac{1}{|h - t_i|},\ 128\right)$$

여기서 $t_i$는 그룹 $i$의 대표적 높이입니다. 거리가 0이면 0으로 나누는 것을 방지하기 위해 가중치는 128로 고정됩니다. 모든 가중치를 정규화한 후:

- **일반 스칼라 파라미터**(각도 범위, 감쇠 범위, 최소 가지, 최대 깊이, 잎 임계값, 낙엽 파라미터, 줄기 길이, 가지 기울기): 가중 평균.
- **`줄기/잎 설정`**: 최대 가중치 그룹의 색상 목록을 직접 사용.
- **`분포 곡선`**(각도 분포, 감쇠 분포): 막대별 가중 평균.

하나의 파라미터 그룹만 있으면 혼합 계산을 건너뛰고 해당 그룹의 모든 파라미터를 직접 사용합니다.

#### 7.4 참고 예시

기본 나무 종 `단풍나무`에는 "줄기"와 "수관" 두 개의 파라미터 그룹이 있습니다. 양자의 유일한 차이는 `분기 각도 범위`, `길이 감쇠 범위`, `대표적 유효 높이`입니다.

**"줄기"** 파라미터 그룹:
- 대표적 유효 높이는 `0`이며, 지면에 가까울수록 파라미터 그룹의 가중치가 높아집니다.
- 분기 각도 범위는 `0 ~ 10` 균등 분포로 줄기의 직립 형태를 유지합니다.
- 길이 감쇠 범위는 `0.8 ~ 0.95` 균등 분포로 줄기가 길게 유지되고 하부 가지가 감소합니다.

**"수관"** 파라미터 그룹:
- 대표적 유효 높이는 `180`이며, 기본적으로 지면에서 멀어질수록 파라미터 그룹의 가중치가 높아집니다.
- 분기 각도 범위는 `0 ~ 60`으로 양단의 확률이 높아 수관의 볼륨감 있는 형태를 유지합니다.
- 길이 감쇠 범위는 `0.3 ~ 0.95`로 대단의 확률이 높고 소단의 확률은 낮지만 가능하여, 수관의 층층이 다양한 형태를 유지하고 수관의 다양성을 증가시킵니다.

## 나무 종 가져오기/내보내기

프로그램은 모든 사용자 정의 나무 종을 `AppData\Local\프랙탈 트리 데스크톱 월페이퍼\trees` 폴더의 `<나무 이름>.json` 파일에 저장합니다. 파라미터 설정 창 왼쪽 하단의 `나무 폴더 열기` 버튼을 클릭하여 빠르게 열 수 있습니다.

- 나무 종을 가져오려면 가져오려는 `<나무 이름>.json` 파일을 해당 폴더에 넣고 프로그램을 재시작하면 됩니다.
- 나무 종을 내보내려면 내보내려는 `<나무 이름>.json` 파일을 복사하면 됩니다.

## 렌더링 창

메인 화면에서 임의의 나무 종 버튼을 클릭하거나 편집 화면에서 미리보기를 클릭하여 프랙탈 트리 렌더링 창을 엽니다. 드래그와 확대/축소를 지원합니다.

- **새로고침**: 무작위 시드를 리셋하고 다시 그립니다. 각 클릭마다 다른 형태가 됩니다.
- **이미지로 저장**: 창 크기로 PNG에 렌더링하여 데스크톱에 `<나무 이름>_<타임스탬프>.png`로 저장합니다. 완료 후 창은 닫히지 않습니다.
- **데스크톱 월페이퍼로 설정**: 화면 해상도의 2배로 BMP에 렌더링하여 시스템 API를 호출해 Windows 데스크톱 월페이퍼로 설정합니다. 완료 후 창이 자동으로 닫힙니다.
