A fractal tree generator developed with C# Windows Forms that can generate various forms of fractal trees and set them as Windows desktop wallpapers.
Let's plant trees on your computer desktop!

## Introduction

`Fractal Tree Desktop Wallpaper` is a powerful fractal tree generation tool that, through recursive algorithms and parametric configuration, can simulate almost all types of tree forms in the world.

## Core Features

- **Powerful Parameter System**: Really powerful
- **Preset Tree Library**: No problem if you don't know how to adjust parameters
- **Custom Editing**: Support creating, editing, saving, importing, and exporting custom tree species
- **Multi-language Support**: Supports 8 languages (although except for Chinese, all are AI-translated)
- **One-click Export/Set Wallpaper**: Improve tree planting efficiency
- **Fallen Leaf System**: Full of atmosphere

## Installation

1. Visit the [project release page](https://github.com/youke1686/Fractal-Tree-Desktop-Wallpaper/releases)
2. Download the latest version of the `exe` file matching your device
3. No installation required, just double-click to start

## Quick Start

1. Run the program
2. Select an interesting tree on the main interface and click to view the effect
3. Click "Refresh" to get a different tree
4. Export as image or set as wallpaper
